# @piclaw/pi-tui

Professional TUI components for building AI coding assistants. Built on top of `@mariozechner/pi-tui` and `@mariozechner/pi-coding-agent`.

## Components

- `ChatContainer`: Scrollable chat display with support for user/assistant messages, tool execution, and bash output.
- `CustomEditor`: Enhanced multiline editor with autocomplete, history, and auto‑indent.
- `FooterComponent`: Status line with dynamic items.
- Overlays & selectors: `ModelSelectorComponent`, `ThinkingSelectorComponent`, `SettingsSelectorComponent`, etc.

See the main documentation for usage examples.
