# Changelog

## 1.0.0 (2026-04-24)

### Features
- Initial release of QClaw TUI
- Chat with markdown, code blocks, images
- Editor with autocomplete, history, auto‑indent
- Model selector with thinking level per model
- Session management (list, fork, delete)
- OAuth login flow for multiple providers
- Theme switching (dark/light/auto)
- Settings selector (compaction, images, padding, autocomplete)
- Thinking level selector
- Footer with status, git branch, cwd, token usage
- Expandable tool output with syntax highlighting
- Bash mode (`!` prefix)
- Abort (Ctrl+C), copy (Ctrl+Shift+C), fullscreen (F11)
- Virtual scroll via maxMessages
- Debounced footer updates
- Telemetry and error countdown
